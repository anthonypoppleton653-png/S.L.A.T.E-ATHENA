🐳 Docker Quick-Start

Pull the stable image from GitHub Container Registry and run SLATE in minutes.

STABLE IMAGE v3.0.0
# Pull stable image from GHCR
docker pull ghcr.io/synchronizedlivingarchitecture/slate:stable

# Start full stack (GPU + Ollama + ChromaDB + Grafana)
docker compose up -d

# CPU-only alternative
docker compose --profile cpu up -d

# Open dashboard
open http://127.0.0.1:8080
