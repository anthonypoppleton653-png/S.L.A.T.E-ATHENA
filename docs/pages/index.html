System Requirements

SLATE scales from modest hardware to high-end workstations.

✅ Minimum

  • OS Windows 10/11, Linux, macOS
  • Python 3.11+
  • RAM 8GB
  • GPU None (CPU-only mode)
  • Storage 10GB free

🚀 Recommended

  • OS Windows 11, Ubuntu 22.04+
  • Python 3.11+
  • RAM 16GB+
  • GPU NVIDIA RTX 20xx+ (8GB+ VRAM)
  • Storage 50GB+ SSD
