## SLATE Copilot SDK Plugin

SLATE provides a full GitHub Copilot SDK plugin at `slate/slate_copilot_sdk.py` with:
- **15 registered tools** for system management, GPU control, workflow orchestration
- **ActionGuard hooks** for security validation (PreToolUse, PostToolUse, UserPromptSubmit)
- **5 skills** (status, runner, orchestrator, workflow, help)
- **MCP server** at `slate/mcp_server.py` with 12 stdio tools
- **Custom agent** "slate" for autonomous operations

## Security Rules

- All bindings: `127.0.0.1` only — never `0.0.0.0`
- ActionGuard blocks: `rm -rf`, `eval()`, `exec()`, `0.0.0.0`, `base64.b64decode`
- PII scanner detects: API keys, tokens, credentials, emails
- No `curl.exe` — use Python `urllib.request` or PowerShell `Invoke-RestMethod`

## Quick Start

```bash
# Check plugin status
python slate/slate_copilot_sdk.py --status

# Execute a tool directly
python slate/slate_copilot_sdk.py --tool slate_status --args '{"format": "quick"}'

# Register skills
python slate/slate_copilot_sdk.py --register-skills

# Run the full plugin
python slate/slate_copilot_sdk.py

# Verify installation
python slate/install_copilot_sdk.py --verify
```
